# Localization Glossary

This is the single source of truth for user-facing terminology until broader localization work is requested.

## Current Product Language

- English-first only
- No Arabic copy shipped yet
- No RTL layout contract shipped yet

## Usage Rules

- Reuse existing wording from `src/lib/strings.ts` when possible.
- Do not introduce synonyms for the same control without a clear product reason.
- Keep support replies consistent with the user guides.
- If localization expands later, update this glossary first and then update app copy in the smallest affected scope.
